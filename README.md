# AceFX
Ace Code Editor Wrapper For JavaFX

See https://ace.c9.io/


Add to your project:

Gradle:
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
	dependencies {
	        implementation 'com.github.orange451:AceFX:Tag'
	}
```


Maven:
```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
  
	<dependency>
	    <groupId>com.github.orange451</groupId>
	    <artifactId>AceFX</artifactId>
	    <version>Tag</version>
	</dependency>
```
